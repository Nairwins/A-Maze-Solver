# Maze Generator & Solver

A Python project that generates random mazes using **Prim's algorithm** and solves them using the **A\* algorithm**. It also features a **Pygame UI** to visualize the maze, start/goal positions, and the computed path.

---

## Features

- Procedural **maze generation** using Prim's algorithm.
- **Randomized start and goal positions**.
- **A\* pathfinding** to find the shortest path in the maze.
- **Interactive Pygame UI**:
  - Randomize start/goal positions
  - Run A* pathfinding
- Optionally **read maze from a text file**.
